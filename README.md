# 💛 YukNikah — Generator Link Undangan Massal

YukNikah Url Generator adalah website sederhana untuk membuat link undangan personal secara massal langsung dari file Excel, CSV, atau TXT tanpa backend.

Cukup upload daftar nama tamu → generate otomatis → copy atau export hasilnya.

---

## ✨ Fitur Utama

- 📂 Import file:
  - Excel (.xlsx)
  - CSV (.csv)
  - TXT (.txt)

- 🔗 Generate link undangan otomatis

- 📋 Copy link per tamu

- 📑 Copy semua link sekaligus

- 📤 Export hasil:
  - Excel (.xlsx)
  - TXT (.txt)

- 🔍 Search & filter nama tamu

- 📊 Statistik jumlah data

- 📱 Responsive design

- ⚡ 100% berjalan di browser (tanpa backend)
